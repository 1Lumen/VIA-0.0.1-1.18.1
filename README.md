# VIA

Vanilla Improvements and Additions aims to imrove on the vanilla game by adding and changing various aspects of Minecraft.
This mod is still in early development! Requires Geckolib and Fabric API!
## Currently this Mod adds
* Parrotfish excrete sand after eating coral blocks or coral fan blocks
* copper block recipe changed from 3x3 to 2x2
## Planned Features
* Parrotfish variants
* New brewing system
* Wandering Trader rework
* Villager dropping their inventory on death
* Villagers pathfinding to emeralds
* new Azalea tree type
* option for fog in swamps
* knockback from eggs and snowballs
* some other recipe changes like potions of turtlemaster
